# 我的个人博客

## React hooks + Next.js + Egg.js 博客系统


### 系统介绍

这是一个适合写IT技术文章的博客系统，主要使用的技术是React Hooks + Next.js + Egg.js + MySql + Ant Design 。

#### 部署涉及技术栈： 

<ul>
<li>Liunx相关操作，比如最简单的基础操作。</li>
<li>Linux下搭建MySql和Nginx。</li>
<li>Linux下进行内网穿透。</li>
<li>PM2的相关使用。</li>
<li>Linux相关安全设置，比如开端口，关端口，自动监控这些。</li>
</ul>
